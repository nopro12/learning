# Classup - The Learning App 🌟

A modern learning platform that enables educators and students to create meetings, collaborate with multi-language support, and share knowledge with others.

## Features

- 🎯 **Create & Join Meetings** - Host or join learning sessions
- 🌍 **Multi-Language Support** - Support for English, Tiếng Việt, and Español
- 👥 **Real-time Collaboration** - Work together with others in real-time
- 📤 **Easy Sharing** - Share content with your learning community
- 🚀 **Deploy to Free Domain** - Host on Vercel with a free domain

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: CSS3
- **Deployment**: Vercel

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/nopro12/learning.git
cd learning
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Building for Production

```bash
npm run build
npm run start
```

## Deployment on Vercel

The easiest way to deploy is using [Vercel](https://vercel.com):

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project" and select this repository
4. Vercel will automatically detect it's a Next.js project
5. Click "Deploy"

Your app will be live on a free Vercel domain within seconds! 🚀
